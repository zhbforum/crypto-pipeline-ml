# Аналіз ARIMA-моделі для щоденних логарифмічних доходностей BTC/USDT

## 1. Вступ

Для моделювання часової структури доходностей BTC/USDT було використано
автоматичний підбір ARIMA-моделі (`auto_arima`) на логарифмічних щоденних доходностях
за 2017–2025 роки.

Попередній описовий аналіз показав:
- низьку середню доходність (≈0.001),
- високий рівень волатильності,
- сильну ліву асиметрію (skew ≈ –1),
- надлишкову куртозис (kurtosis ≈ 18),
що вказує на складну часову структуру та «товсті хвости».

---

## 2. Підібрана ARIMA-модель

`auto_arima` обрав модель **ARIMA(0,0,2)** (MA(2)) з константою:

| Параметр | Оцінка | P-value |
|---------|--------|--------|
| intercept | 0.0010 | 0.138 |
| ma(1) | –0.0493 | 0.000 |
| ma(2) | +0.0456 | 0.003 |
| sigma² | ≈0.0013 | <0.001 |

Модель статистично значуща за MA-параметрами, що свідчить про наявність
короткострокової структури залежностей у доходностях.

AIC: **−11484.69**

---

## 3. Діагностика залишків

- Ljung-Box p-value ≈ 0.99 → залишки **не мають автокореляції**  
- Skew: –0.99 → **ліва асиметрія**
- Kurtosis: 18.6 → **надлишкова кучність (fat tails)**  

Модель добре описує внутрішню динаміку, але залишки залишаються ненормальними —
типово для фінансових рядiв.

---

## 4. Прогноз ARIMA

Побудовано прогноз на 14 днів уперед з 95% довірчим інтервалом.

![ARIMA Forecast](../data/arima_forecast.png)

Прогноз показує:
- середня очікувана доходність знаходиться близько до нуля,
- довірчий інтервал відносно вузький,
- сильних короткострокових трендів модель не передбачає.

---

## 5. Висновки

1. Доходності BTC добре описуються MA(2)-структурою.
2. Існує слабка, але статистично значуща залежність доходності від шумових компонентів минулих днів.
3. Залишки моделі мають властивості «білих шумів», що підтверджує коректність структури ARIMA.
4. Модель не передбачає значних трендів, що узгоджується з властивістю **відсутності довгострокової автокореляції в доходностях криптовалют**.
5. Прогностичний інтервал демонструє, що короткострокові зміни важко прогнозувати —
   BTC поводиться переважно як стохастичний процес із високою волатильністю.

ARIMA-модель є коректною та адекватною для часових рядів лог-доходностей.
