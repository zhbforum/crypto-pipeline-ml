# Узагальнюючі висновки щодо описового статистичного аналізу BTC/USDT

## 1. Вихідні дані
Для аналізу використано щоденні логарифмічні доходності Bitcoin (BTC/USDT) 
за період **2017–2025 рр.**, отримані з відкритого API Binance.
Загальний обсяг вибірки: **3026 спостережень**, що повністю відповідає вимогам ТЗ.

## 2. Основні статистичні характеристики

- **Середня денна доходність:** 0.001009  
  (≈ 0.10% на день; свідчить про позитивний середньостроковий тренд зростання).

- **Дисперсія:** 0.001318  
- **Стандартне відхилення:** 0.03631  
  (≈ 3.63% на день; Bitcoin є високоволатильним активом).

- **Асиметрія (skewness):** –0.9687  
  Розподіл скошений вліво, тобто різкі падіння трапляються частіше та є сильнішими, ніж різкі зростання.

- **Куртозис:** 15.54  
  Значення значно перевищує 0 (нормальний розподіл).  
  Розподіл має «товсті хвости» (leptokurtic), що характерно для фінансових ринків з частими екстремальними подіями.

## 3. Довірчий інтервал середньої доходності

95% ДІ для середньої денний доходності:  
**[-0.000284; 0.002303]**

Тобто середнє значення з високою ймовірністю знаходиться між –0.028% та 0.230% на день.
Це означає, що середній тренд додатній, але коливання значні.

## 4. Інтерпретація

- Bitcoin демонструє високий рівень нестабільності.
- Негативна асиметрія свідчить про підвищений ризик різких падінь.
- Високий куртозис підтверджує можливість появи «чорних лебедів» — екстремальних днів із надзвичайними прибутками або збитками.
- Попри високу волатильність, середня доходність лишається позитивною.
